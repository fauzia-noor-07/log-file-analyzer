# log-file-analyzer
Python script jo log files ko analyze karke errors detect karta hai.

**Features:**
- Total lines count karta hai
- ERROR/FAILED wali lines find karta hai 
- Top 3 error types with count batata hai
- Summary report.txt mein save karta hai

**How to run:**

```python main.py```

Sample Output:
```
Total lines: 6
Total errors: 3
Top error types:
    TIMEOUT : 2 times
```
